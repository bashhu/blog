Devops是ITIL服务管理结合当前IT公司情况制定的一套行之有效的管理操作流程，它也可以落地实施成一套高效紧凑的工作系统。

当前我们把devops简单分为：
- 持续集成(CI): 包括任务发布，任务触发，任务关联，自动化实施
- 持续交付(CD)：单元测试，需求及bug管理，数据监控，数据展示
